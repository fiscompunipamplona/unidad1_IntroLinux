Gráficos:¶

  • Vamos usar Pylab, un paquete que hace parte de uno ms grande llamado matplotlib.
  • Dos tipos de grficos: lineales, dispersin, y grficos de densidad.
In [2]:
from pylab import plot, show

y = [1., 2.4, 1.7, 0.3, 0.6, 1.8]

plot(y)
show()

En este caso Python asign la escala del eje X.
Para especificar esta escala:

In [3]:
x = [0.5, 1., 2., 4., 7., 10.]

plot(x,y)
show()

Con la opcin plot() podemos insertar varios plot en el mismo grfico.

Para salvar el grfico en un archivo:

In [4]:
from pylab import savefig
plot(x, y)
savefig('destination_path.eps', format='eps', dpi=1000)
show()

Para graficar una funcin conocida, por ejemplo la funcin Sin:

In [5]:
from numpy import linspace, sin, cos

x = linspace(0, 10, 1000)
y = sin(x)

plot(x,y)
show()

Note que en este caso, la funcin sin actu sobre cada elemento del array x. De manera equivalente a map(sin, x)

La escala de los ejes se puede modificar¶

In [6]:
from pylab import ylim, xlim, grid
x = linspace(0, 10, 1000)
y = sin(x)

plot(x,y)
ylim(-1.1, 1.1)
xlim(0., 10)
grid("on")
show()
  • Tambin podemos insertar las etiquetas de los ejes
In [7]:
from pylab import xlabel, ylabel
plot(x,y)
ylim(-1.1, 1.1)
xlim(0., 10)
grid("on")
xlabel("Eje X")
ylabel("Eje Y")
show()
